A page with links to all your Pow apps.

Links are relative to the current domain, so you can get quick access to apps on a network device if you're using xip.io.

Use

Jump on your iPhone, iPad, or virtual machine and go to powapps.10.0.0.1.xip.io (sub in the IP to your host's IP).

Add to your home screen, or bookmark!

Installation

Clone somewhere and link in pow:

$ git clone https://github.com/daz/powapps
$ cd ~/.pow
$ ln -s /path/to/powapps
